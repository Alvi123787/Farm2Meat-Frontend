# MeatByAlvi Frontend

MeatByAlvi helps customers browse healthy bakray, place orders, and stay connected with trusted service.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Context API
- Framer Motion
- React Icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
